# Bugs and Errors Log

- **Issue**: Unable to click "Create New Room" button on the home page.
- **Status**: RESOLVED

## Runtime Errors
- **Issue**: Hydration error in `layout.tsx` due to browser extension attributes.
- **Status**: RESOLVED
- **Fix**: Added `suppressHydrationWarning` to the `<html>` tag in `client/app/layout.tsx`.

- **Issue**: `ReferenceError: module is not defined` in `RoomClient.tsx`.
- **Status**: RESOLVED
- **Fix**: Added `typeof module !== 'undefined'` check before accessing `module.hot`.

- **Issue**: `ReferenceError: module is not defined` in `PlayerComponent.tsx` causing video player crash.
- **Status**: RESOLVED
- **Fix**: Removed all unsafe `module.hot` code and replaced with safe browser check `if (typeof window !== "undefined")`.
